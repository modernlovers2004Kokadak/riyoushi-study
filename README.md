# 理容師国家試験対策 統合版 v2.0.2

GitHub Pages にアップロードするだけで利用できる完成版です。

## 反映内容

- フローティングホームボタンを追加
- ランク表示を小さな色丸付きの「最重要」「重要」「頻出」の縦積みに統一
- 数字を半角数字に統一
- 文字化けしないフォルダ名に整理
- トップページ、教材集、問題集の導線を維持

## 使い方

このZIPを展開し、中身をGitHubリポジトリにアップロードしてください。
