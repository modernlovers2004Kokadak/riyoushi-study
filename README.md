# 理容師国家試験対策 ９法令学習

GitHub Pages にそのまま公開できる完成版です。

## アップロードするファイル

- index.html
- style.css
- script.js
- data.js
- manifest.json
- sw.js

## 公開方法

１．ZIPを解凍  
２．中身のファイルをすべてGitHubにアップロード  
３．Settings → Pages → Branch を main / root に設定  
４．公開URLを開く  

## 機能

- 約６００条文をカード化
- アコーディオン表示
- 検索機能
- 重要度表示
- 学習済み保存
- スマホ最適化
- オフライン対応
